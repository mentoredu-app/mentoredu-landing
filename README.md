# MentorEdu Landing Page

Página de presentación estática de la plataforma MentorEdu. Desplegada automáticamente en GitHub Pages con cada push a `main`.

## Stack

- HTML · CSS · JavaScript
- GitHub Actions (CI/CD)
- GitHub Pages (hosting)

## Versión desplegada

`v1.0.0` — [ver tag](https://github.com/mentoredu-app/mentoredu-landing/releases/tag/v1.0.0)

## Live

https://mentoredu-app.github.io/mentoredu-landing/
